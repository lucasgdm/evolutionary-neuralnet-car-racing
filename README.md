## Instructions
Install swig (needed for Box2D to work)  
`$ apt install swig`

Install pip dependencies (preferably using a virtual environment)  
`$ pip install -r requirements.txt`

Run  
`$ python3 main.py`

...Or play against it  
`$ python3 play_against.py`
